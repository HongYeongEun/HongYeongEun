## 🧩 Side Projects


### 🎲 [반려동물 쇼핑 & 기부 플랫폼, 멍냥꽁냥](https://github.com/suikari/Myky)

→ Spring Boot + JSP + Vue.js + KakaoMap API 기반

→ React + Node.js + MySQL 기반

→ Java + Spring Framework + Kakao Map API 기반
→ 위치 기반 거리 계산 및 마커/팝업 구현 포함
